# Quiz_App
### Run Locally

Clone the project

```bash
  https://github.com/arunkr741/Quiz_App.gi\
```

Go to the project directory
```bash
  Frontend
  cd Front-end
  Backend
  cd Back-end
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
  ```
## Tech Stack

- Client: React, Redux 

- Server:  Node, Express

- DataBase: MongoDB
